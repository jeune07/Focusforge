# Overview

Focus Force is a web app designed to assist people in planning their daily tasks.

**Description:**
Focus Force allows users to create tasks, delete tasks, mark them as complete, and each task has a creation date.

**Purpose:**
The purpose of creating this software is to help users track their daily tasks.

# Development Environment

**Tools Used:**
- React.js: Utilized `useState` to manage some state and `useReducer` for custom state management in forms.
- Styling: Bootstrap 5 and CSS.
- Programming Language: JavaScript.
